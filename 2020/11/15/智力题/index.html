计算机应用数学 —– 每周智力题

老鼠喝药

  • 已知有 11 瓶无色无味的药水，其中有 1 瓶有毒并且其他 10 瓶无毒，喝了有毒的药水后会在 23 个小时之后的一个小时内毒发身亡（死亡时间在 1 个小时内随机）。现在有 4 只老鼠，如何在一天之内 (第二天) 找出这瓶有毒的药水？

以二进制计数方式对 11 瓶药编号（如下）：

  1. 第一瓶药 ： 0 0 0 1
  2. 第二瓶药 ： 0 0 1 0
  3. 第三瓶药 ： 0 0 1 1
  4. 第四瓶药 ： 0 1 0 0
  5. 第五瓶药 ： 0 1 0 1
  6. 第六瓶药 ： 0 1 1 0
  7. 第七瓶药 ： 0 1 1 1
  8. 第八瓶药 ： 1 0 0 0
  9. 第九瓶药 ： 1 0 0 1
  10. 第十瓶药 ： 1 0 1 0
  11. 第十一瓶药： 1 0 1 1

四只老鼠分别按照 “1” 对应的位置喝药即可。
